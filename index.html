This is an =APIs.json driven client code generator. If you have any Swagger 2.0 files, that are indexed by an APIs.json, you can enter the URL below, and generate seven client language libraries in PHP, Python, Ruby, Java, C#, Objective-C, and Android Java, for all API endpoints defined.

The script is free to use, and doesn't require registration to use, but depending on how large the API is, it can take 30 to 60 seconds to run. Depending on volume, we may scale up the server, and if you'd like a dedicated server let me know.

A default APIs.json is provided to demonstrate how it works, click "view" below, and you can see what is considered a minimum viable APIs.json file.

APIs.json codegen is in beta, and still be hardened to handle more scenarios. It uses Swagger codegen as its backend, and if there is a language or framework you'd like to see added to the stack, please submit an issue, with details.
